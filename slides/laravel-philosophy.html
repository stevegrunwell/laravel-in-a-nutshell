Laravel Philosophy

"Laravel aims to make the development process a pleasing one for the developer without sacrificing application functionality. Happy developers make the best code."

Developer Candy

  • Requires PHP 5.4+
  • ActiveRecord-esque Eloquent ORM
  • Leverages the best third-party libraries (via Composer):
    • Symfony components for HTTP, routing, translation, and more
    • Carbon for datetime parsing
    • github.com/laravel/framework
  • Handlebars-inspired templating system, Blade
  • Built-in support for PHPUnit
